# 上海2022年新冠疫情文章及视频存档 covid-19-in-shanghai-archive 
本repo试图记录所有被全网删除的文章、评论、视频、图片。 This repository is a collection of documents or records about what happened in shanghai during the 2022 COVID-19 breakout.
